# aws-azure-cloud-projects
# Cloud Infrastructure Projects (AWS & Azure)

This repository contains hands-on cloud infrastructure projects built using
Infrastructure as Code (Terraform and Azure Bicep) and automated using GitHub Actions.

## Technologies
- AWS (VPC, EC2, ECS, IAM, CloudWatch)
- Azure (VNet, VM, Container Apps, Azure Monitor)
- Infrastructure as Code: Terraform, Bicep
- CI/CD: GitHub Actions

## Purpose
These projects demonstrate production-style cloud engineering practices including:
- Infrastructure as Code
- Automation and CI/CD
- Security-by-design
- High availability and scalability
